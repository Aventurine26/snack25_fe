## Requirements

- PNPM을 패키지 매니저로 사용할 것이므로 아래 명령어를 통해 전역 설치

```bash
npm i -g pnpm
```

### 사용방법

- 스크립트 실행 시 `npm run` 자리에 `pnpm`을 사용해주세요.

```bash
npm run dev   -> pnpm dev
npm run start -> pnpm start
npm run build -> pnpm build
npm run lint  -> pnpm lint
...
```
